# nmap2json

nmap2json converts a NMap XML file to a JSON file.

## Command Line Options

	Usage: ./nmap2json [OPTIONS] [FILE] ([FILE] [FILE]...)
	  -outdir string
	    	Output Directory. (default ".")
	  -prettify
	    	Prettify JSON Output.
